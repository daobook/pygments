欢迎！¶

这里是 Pygments 主页。它是一个通用的语法高亮器，适用于代码托管、论坛、维基或其他需要美化源代码的应用程序。高亮有：

  • 支持超过 500 种语言和其他文本格式的广泛内容

  • 特别注意细节，增加高亮质量

  • 对新的语言和格式的支持很容易添加；大多数语言使用简单的基于正则表达式的词法机制

  • 有多种输出格式可供选择，其中包括 HTML、RTF、LaTeX 和 ANSI 序列

  • 它可以作为一个命令行工具和一个库使用

在 常见问题清单 或 文档 中阅读更多信息，或者 下载最新版本。

贡献¶

像每一个开源项目一样，我们一直在寻找志愿者来帮助我们进行编程。需要有 Python 知识，但不要害怕：Python 是一种非常清晰和容易学习的语言。

开发是在 GitHub 上进行的。

如果你发现了一个错误，只需在 GitHub 跟踪器上开一个 ticket。请务必登录，以便在问题被修复时得到通知 —— 由于该库相当稳定，开发速度并不快。你也可以给开发人员发送电子邮件，见下文。
