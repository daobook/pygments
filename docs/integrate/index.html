在各种场景下使用 Pygments¶

Markdown¶

从Pygments 0.9开始，发行版提供了 Markdown 预处理器的样本代码，它使用 Pygments 来渲染 external/markdown-processor.py 中的源代码。你可以根据自己的喜好复制和改编它。

TextMate¶

Antonio Cangiano 为 TextMate 创建了一个 Pygments 捆绑包，可以通过一个简单的菜单选项来给代码着色。它可以在 这里 找到。

Bash 自动补全¶

源码发行包含一个文件 external/pygments.bashcomp，为 bash 中的 pygmentize 命令设置完成。

用于其他语言的封装器¶

这些库为 Python 以外的其他语言的用户提供 Pygments 高亮显示：

  • pygments.rb, a pygments wrapper for Ruby

  • Clygments, a pygments wrapper for Clojure

  • PHPygments, a pygments wrapper for PHP
