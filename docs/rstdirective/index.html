在 ReST 文件中使用 Pygments¶

许多 Python 人使用 ReST 来记录他们的源代码、程序、脚本等等。这也意味着文档通常包括源代码样本或片段。

你可以使用一个自定义指令轻松地为你的 ReST 文本启用 Pygments 支持 —— 这也是本文档显示源代码的方式。

从 Pygments 0.9 开始，该指令以 external/rst-directive.py 的形式在发行版中提供。你可以根据自己的喜好复制和改编这段代码。
